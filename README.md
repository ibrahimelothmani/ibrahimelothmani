## 🧑‍💻 Engineering Philosophy
**"I build software with the precision of a structural engineer and the agility of a DevOps architect."**

My background in **Civil Engineering** serves as the blueprint for my software career. It taught me that the most beautiful systems are those that are resilient under pressure, maintainable over decades, and designed with safety as a core component—not an afterthought.

Today, I translate that structural rigor into **Cloud-Native environments**, focusing on:
- **Resilience:** Designing fault-tolerant, multi-tenant SaaS architectures.
- **Security:** "Shift-left" DevSecOps, automated hardening, and compliance-driven CI/CD.
- **Efficiency:** Optimizing resource allocation from the hardware edge (IoT) to the cloud (K8s).
